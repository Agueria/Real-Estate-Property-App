# Real estate property App

You can create houses in strapi.

## Install Node Modules
>In Terminal
>cd ./frontend
>npm -i
>cd ..
>cd ./backend
>npm -i

## Run The App
In Terminal
cd ./backend 
npm run dev
Then open a new terminal
cd ./frontend
npm run start
