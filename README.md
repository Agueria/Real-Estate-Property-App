# Real estate property App

You can create houses in strapi.

## Install Node Modules
In Terminal <br />
cd ./frontend <br />
npm -i <br />
cd .. <br />
cd ./backend <br />
npm -i <br />

## Run The App
In Terminal <br />
cd ./backend <br />
npm run dev <br />
Then open a new terminal <br />
cd ./frontend <br />
npm run start <br />
